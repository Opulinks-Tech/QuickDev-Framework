=========================================================================
 Opulinks SDK add into QuickDev_Framework manual
=========================================================================

Please follow steps to add the Opulinks SDK into QuickDev_Framework framework

1. Find and downlod the SDK from github https://github.com/Opulinks-Tech

    For example: if wish to using latest OPL1000A3-SDK, find the repository of "OPL1000A3-SDK" at github
                 and download the whole package data.


2. In SDK root folder, copy "FW_Pack", "SDK", "Tool" folders to QuickDev_Framework\sdk folder

    At final view in QD_FWK\sdk like below:

        QuickDev_Framework\sdk\FW_Pack
        QuickDev_Framework\sdk\SDK
        QuickDev_Framework\sdk\Tool


3. Start program example in QuickDev_Framework\example
